# b3-tools

Usage:
 * monitor.sh
 * watch -n <interval> -d ./monitor.sh (for auto refresh every <interval> seconds)

Requirements:
 * jq (json parser) https://stedolan.github.io/jq/download/

Prints information about your B3 Fundamental Node
